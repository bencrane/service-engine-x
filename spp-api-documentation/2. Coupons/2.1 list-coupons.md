# Coupons - List all Coupons

## List all coupons

**GET**
`https://{workspaceUrl}/api/coupons`

Return a list of Coupon objects. The coupons are sorted by creation date with the most recent coupons appearing first. You can filter and sort the results using query parameters.

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| limit | integer | Items per page |
| page | integer | Page number |
| sort | string | Sort by fields |
| filters[id][$in][] | array[integer] | Example filter on user_id. You can filter by any other field or operator and combine multiple filters. Supported operators include $eq (=), $lt (<), $gt (>), $in (in array, requires an array of values). |

Examples:
- limit: `10`
- page: `1`
- sort: `id:desc`

### Headers

| Header | Type | Description |
|--------|------|-------------|
| X-Api-Version | string | API version identifier, in case of any issues with API please provide this identifier when contacting support |

Example: `2024-03-05`
