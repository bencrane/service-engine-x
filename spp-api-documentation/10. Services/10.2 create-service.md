# Services - Create a Service

Create a service

**POST**
`https://{workspaceUrl}/api/services`

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Headers

| Parameter | Type | Description | Example |
|-----------|------|-------------|---------|
| X-Api-Version | string | API version identifier, in case of any issues with API please provide this identifier when contacting support | `2024-03-05` |

### Body

`application/json`

| Parameter | Type | Required | Description | Default | Example |

| name | string | required | Name | | `Service name` |
| description | string | | Description | | `Service description` |

| currency | string | required | Currency | | `USD` |

| r_period_t | string | | R period t | | `D` |
