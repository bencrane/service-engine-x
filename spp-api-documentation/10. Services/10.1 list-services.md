# Services - List all services

List all services

**GET**
`https://{workspaceUrl}/api/services`

Return a list of Service objects. The services are sorted by creation date in descending order.

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Query Parameters

| Parameter | Type | Description | Example |
|-----------|------|-------------|---------|
| limit | integer | Items per page | `10` |
| page | integer | Page number | `1` |
| sort | string | Sort by fields | `id:desc` |
| filters[id][$in][] | array[integer] | Example filter on user_id. You can filter by any other field or operator and combine multiple filters. Supported operators include $eq (=), $lt (<), $gt (>), $in (in array, requires an array of values). | |

### Headers

| Parameter | Type | Description | Example |
|-----------|------|-------------|---------|
| X-Api-Version | string | API version identifier, in case of any issues with API please provide this identifier when contacting support | `2024-03-05` |
