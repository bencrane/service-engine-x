# Order Tasks - Create a task for an order

Create a task for an order

**POST**
`https://{workspaceUrl}/api/order_tasks/{order}`

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| order | string or allOf | required | The order to create a task for. |

Example:
`E4A269FC`

### Body

`application/json`

#### Task request

| Parameter | Type | Required | Description | Example |
|-----------|------|----------|-------------|---------|
| employee_ids | array[integer] | | Employee IDs assigned to the task | |
| name | string | required | Task name | `Design Logo` |
| description | string | | Task description | `Create a new logo for the client` |
| sort_order | integer | | Sort order | `1` |
| is_public | boolean | | Indicates whether the task is visible on the portal side. | `true` |
| for_client | boolean | | If true, clients are allowed to complete this task. Mutually exclusive with employee_ids. | `false` |
| deadline | integer | | Deadline in hours. For days use 24-hour increments. Mutually exclusive with `due_at`. | `7` |
| due_at | string\<date-time\> | | Due date | `2024-12-31T23:59:59+00:00` |
