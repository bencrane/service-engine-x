# Team - List all team members

List all team members

**GET**
`https://{workspaceUrl}/api/team`

Return a list of User objects. Team members are sorted by creation date in descending order.

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Query Parameters

| Parameter | Type | Description | Example |
|-----------|------|-------------|---------|
| limit | integer | Items per page | `10` |
| page | integer | Page number | `1` |
| sort | string | Sort by fields | `id:desc` |
| filters[user_id][$in][] | array[integer] | Example filter on user_id. You can filter by any other field or operator and combine multiple filters. Supported operators include $eq (=), $lt (<), $gt (>), $in (in array, requires an array of values). | |

### Headers

| Parameter | Type | Description | Example |
|-----------|------|-------------|---------|
| X-Api-Version | string | API version identifier, in case of any issues with API please provide this identifier when contacting support | `2024-03-05` |
