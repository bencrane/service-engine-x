# Filled Form Fields - List all filled form fields

## List all filled form fields

**GET**
`https://{workspaceUrl}/api/form_data`

Return a list of FilledFormField objects for a specific order or ticket. Either order or ticket parameter must be provided, but not both.

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| order | any | Order ID, if provided will return only filled form fields for this order, can't be used with ticket parameter |
| ticket | any | Ticket ID, if provided will return only filled form fields for this ticket, can't be used with order parameter |

### Headers

| Header | Type | Description |
|--------|------|-------------|
| X-Api-Version | string | API version identifier, in case of any issues with API please provide this identifier when contacting support |

Example: `2024-03-05`
