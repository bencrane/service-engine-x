# Tickets - Update a Ticket

Update a ticket

**PUT**
`https://{workspaceUrl}/api/tickets/{ticket}`

Updates a specified ticket by setting the values of the parameters passed. Any parameters not provided will be left unchanged.

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| ticket | string or allOf | required | The ticket to update. |

Example:
`E4A269FC`
