# Tickets - List all messages for the ticket

List all messages for the ticket

**GET**
`https://{workspaceUrl}/api/ticket_messages/{ticket}`

Return a list of Messages for the ticket. The messages are sorted by creation date in descending order.

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| ticket | string or allOf | required | The ticket to get messages for. |

Example:
`E4A269FC`
