# Clients - Retrieve a Client

## GET

`https://{workspaceUrl}/api/clients/{user}`

Retrieve a User object.

---

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| user | integer or allOf | required | The client to get. Example: `1` |

### Headers

| Header | Type | Description |
|--------|------|-------------|
| X-Api-Version | string | API version identifier, in case of any issues with API please provide this identifier when contacting support. Example: `2024-03-05` |

### Pagination Links

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| first | string | required | First. Example: `https://example.spp.co/api/{resource}?page=1` |
| last | string | required | Last. Example: `https://example.spp.co/api/{resource}?page=3` |
| prev | string or null | required | Prev. Example: `null` |
| next | string or null | required | Next. Example: `https://example.spp.co/api/{resource}?page=2` |
