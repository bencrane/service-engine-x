# Clients - List all Clients

## GET

`https://{workspaceUrl}/api/clients`

Return a list of User objects. Clients are sorted by creation date in descending order.

---

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| expand[] | array[string] | Expand fields |
| limit | integer | Items per page. Example: `10` |
| page | integer | Page number. Example: `1` |
| sort | string | Sort by fields. Example: `id:desc` |
| filters[id][$in][] | array[integer] | Example filter on id. You can filter by any other field or operator and combine multiple filters. Supported operators include $eq (=), $lt (<), $gt (>), $in (in array, requires an array of values). |

### Headers

| Header | Type | Description |
|--------|------|-------------|
| X-Api-Version | string | API version identifier, in case of any issues with API please provide this identifier when contacting support. Example: `2024-03-05` |
