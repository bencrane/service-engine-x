# Tags - List all tags

List all tags

**GET**
`https://{workspaceUrl}/api/tags`

Return a list of Tag objects. Tags are sorted by creation date in descending order.

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Headers

| Parameter | Type | Description | Example |
|-----------|------|-------------|---------|
| X-Api-Version | string | API version identifier, in case of any issues with API please provide this identifier when contacting support | `2024-03-05` |

## Responses

### 200

Tags

#### Headers

| Parameter | Type | Required | Description | Example |
|-----------|------|----------|-------------|---------|
| X-Api-Identifier | string | required | Unique identifier for the request, in case of any issues with API please provide this identifier when contacting support | `efc995d2-bfdb-3bc9-80a3-e828bb9c83c2` |
| X-Api-Version | string | required | API version, in case of any issues with API please provide this identifier when contacting support | `20231214` |

#### Body

`application/json`

##### ApiTagResource

array of:

| Parameter | Type | Required | Description | Example |
|-----------|------|----------|-------------|---------|
| id | integer | required | Tag ID | `1` |
| name | string | required | Name | `Tag name` |
| created_at | string\<date-time\> | required | Created at | `2021-09-01T00:00:00+00:00` |
| updated_at | string\<date-time\> | required | Updated at | `2021-09-01T00:00:00+00:00` |
