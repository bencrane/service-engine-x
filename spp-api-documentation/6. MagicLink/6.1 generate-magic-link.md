# Magic Link - Generate a magic link

## Generate a magic link

**GET**
`https://{workspaceUrl}/api/login_links/{user}`

Generates a magic link for the specified user. The user must be client user, not staff. Users that have 2FA enabled cannot use magic links. Magic link is valid for 2 hour. If the user has a previously generated magic link, it will be reused.

## Request

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| user | integer or allOf | required | The client user to generate a magic link for. |

Example: `1`
