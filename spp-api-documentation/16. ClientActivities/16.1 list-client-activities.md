# Client Activities - List all activities for a client

## GET

`https://{workspaceUrl}/api/clients/{user}/activities`

Return a list of activities for a specific client. Activities are sorted by creation date in descending order.

---

## Request

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer 123`

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| user | integer or allOf | required | The client ID. Example: `1` |

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| page | integer | Page number. Example: `1` |
| limit | integer | Number of results per page. Default: `20` |

### Headers

| Header | Type | Description |
|--------|------|-------------|
| X-Api-Version | string | API version identifier, in case of any issues with API please provide this identifier when contacting support. Example: `2024-03-05` |

---

## Responses

### 200 - Client activities

**Body:** `application/json`

#### data
`array[Activity]` - Activities

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| id | integer | required | Activity ID. Example: `1` |
| content | string | required | Activity content. Example: `Client called to discuss project requirements.` |
| created_at | string\<date-time\> | required | Date created. Example: `2025-12-19T10:00:00.000000Z` |
| updated_at | string\<date-time\> | required | Date updated. Example: `2025-12-19T10:00:00.000000Z` |
| author | object | required | User who created the activity |

#### links
Pagination Links

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| first | string | required | First. Example: `https://example.spp.co/api/{resource}?page=1` |
| last | string | required | Last. Example: `https://example.spp.co/api/{resource}?page=3` |
| prev | string or null | required | Prev. Example: `null` |
| next | string or null | required | Next. Example: `https://example.spp.co/api/{resource}?page=2` |

#### meta
Meta

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| current_page | integer | required | Current page. Example: `1` |
| from | integer | required | From. Example: `1` |
| last_page | integer | required | Last page. Example: `3` |
| links | array[object] | required | Links |
| path | string | required | Path. Example: `https://example.spp.co/api/{resource}` |
| per_page | integer | required | Per page. Example: `20` |
| to | integer | required | To. Example: `20` |
| total | integer | required | Total. Example: `43` |

##### links array object

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| url | string or null | required | URL. Example: `null` |
| label | string | required | Label. Example: `Previous` |
| active | boolean | required | Active. Example: `false` |

### 401
Unauthorized

### 404
Not Found
